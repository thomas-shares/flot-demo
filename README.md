# flot

Sample of how to use Clojure, ClojureScript, jQuery, http-kit, Flot and Chord.

## Installation

Clone this repo.

## Usage

Run "lein cljsbuild once" and "lein run" and point your browser at localhost:8080 (this can take a little while).
